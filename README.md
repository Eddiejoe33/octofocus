## Problem
- People often forget tasks or deadlines.
- Hard to prioritize when everything feels urgent.
- Manual tracking is tedious and scattered.
- Lack of quick overview of tasks.

## Solution
- Input tasks with **name, deadline, and estimated effort**.
- View all tasks in one clear interface.
- Automatically track tasks received in real-time.
- Helps users prioritize and stay focused.
- Reduces the chance of missing deadlines.

## How It Works
1. Open the web app (`index.html`) in a browser.
2. Add a task by entering its name, deadline, and estimated effort.
3. See tasks listed immediately on the page.
4. Tasks received are also printed in real-time in the backend terminal for tracking.

## Tech Stack
- **Frontend:** HTML, CSS (sky.css), JS (script.js)
- **Backend:** Python Flask + Flask-CORS
- **Extra libraries:** Optional Alpine.js for enhancements
